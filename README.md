# Quiz Application using C++

This project is a **simple console-based Quiz Application** developed using **C++** and **Object-Oriented Programming (OOP)** principles.  
It asks users multiple-choice style questions, checks their answers, and displays the final score.

## üßæ Program Description

- The **Question** class stores:
  - A question
  - Its correct answer
- The `ask()` function:
  - Displays the question
  - Takes user input
  - Returns whether the answer is correct
- The **Quiz** class:
  - Maintains the score
  - Controls quiz flow
  - Displays the final result
